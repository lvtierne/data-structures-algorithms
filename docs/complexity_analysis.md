# Complexity Analysis Documentation

## Linked List
- **Append**: O(1)
- **Print List**: O(n)

## Binary Tree
- **Insert**: O(log n) on average
- **In-order Traversal**: O(n)

## QuickSort
- **Best/Average Case**: O(n log n)
- **Worst Case**: O(n^2)

## Binary Search
- **Time Complexity**: O(log n)
